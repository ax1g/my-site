# ax1g.github.io

Hey, this is my little corner of the internet — a portfolio that looks like my Arch Linux desktop because why not?

Built with plain HTML, CSS, and JavaScript. No frameworks, no build tools. Just open `index.html` in a browser and it works.

It's got workspaces (like Hyprland), a command palette (`/` or `Ctrl+K`), dark/light theme (`Ctrl+D`), and a few easter eggs if you type the right things.

I'm **Ankush Gautam**, a backend engineer from Nepal. This site tells you a bit about me, shows some projects I've worked on, and lists ways to reach out.

## Stuff you can do

- Switch workspaces with `1` / `2` or click the dots in the top bar
- Open the command palette with `/` or `Ctrl+K`
- Toggle dark/light theme with `Ctrl+D`
- Press `?` for help
- Type `vim` if you're brave
- Try the Konami code (↑↑↓↓←→←→BA)
- Type `sudo make me a sandwich`

## Tech

Pure HTML + CSS + JS. JetBrains Mono font. Font Awesome icons. Catppuccin theme (Mocha dark / Latte light). GitHub API for the repo list. That's it.
